0. Refer to <MetaLink label="fomo-cli repository" href="https://github.com/igor-26/fomo-cli" isExternal/> for prerequisites and installation.
1. Log in to your Spotify account.
2. Go to <MetaLink label="/dashboard/applications" href="https://developer.spotify.com/dashboard/applications" isExternal/>.
3. Create an app.
4. Copy the following values into the <MetaLink href='https://github.com/igor-26/fomo-cli/blob/master/.env.example#L19' label='.env' isExternal/> file you've created:

   **client_id:**
   Located on /dashboard/applications/app_id.

   **client_secret:**
   Located on /dashboard/applications/app_id.

   **redirect_uri:**
   Redirect URI can be edited by clicking on `Edit Settings`.

5. You should be all set up to run the <MetaLink href='https://github.com/igor-26/fomo-cli#spotify' label='Spotify integration' isExternal/>.
