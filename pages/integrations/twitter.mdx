0. Refer to <MetaLink label="fomo-cli repository" href="https://github.com/igor-26/fomo-cli" isExternal/> for prerequisites and installation.
1. Log in to your Twitter account.
2. Go to <MetaLink label="/portal/dashboard" href="https://developer.twitter.com/en/portal/dashboard" isExternal/>.
3. Create an app.
4. Generate the following keys on `Keys and tokens` tab:
   - **consumer_key**
   - **consumer_secret**
   - **access_token**
   - **access_token_secret**
5. Copy over said values into the <MetaLink label=".env" href="https://github.com/igor-26/fomo-cli/blob/master/.env.example#L29" isExternal/> file you've created.

6. You should be all set up to run the <MetaLink href='https://github.com/igor-26/fomo-cli#twitter' label='Twitter integration' isExternal/>.
